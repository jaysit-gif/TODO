# Simple CLI Todo List

My first-ever coding project that I actually finished and that does something useful! 🎉

A dead-simple command-line todo list built in Python using **Click** for the CLI interface.

## Why I made this
Just for fun! I wanted to build something practical that helps me manage daily tasks without overcomplicating things.

## Features
- Add new tasks
- List all tasks
- Mark tasks as complete
- Delete tasks
